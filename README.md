# Tubes IAE Project

A full-stack application with separate backend and frontend directories.

## Project Structure

```
.
├── backend/          # Backend API (Express + GraphQL)
├── frontend/         # Frontend application (React + Vite)
└── README.md
```

## Setup

### Backend
```bash
cd backend
bun install
bun run seed  # Seed the database
bun run index.js
```

### Frontend
```bash
cd frontend
bun install
bun run dev
```

## Technologies

- **Backend**: Express, GraphQL, SQLite
- **Frontend**: React, Vite, Tailwind CSS
